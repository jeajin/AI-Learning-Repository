- ## Arithmetic operations : 사칙연산과 python 내장 연산

```python
>>> 1 - 2
-1
>>> 4 * 5 - 19

>>> 7 / 5
1.4
>>> 9 // 5 # = [9/5] = round(9/5,0) = round(9/5) = math.floor(9/5)

>>> 3**2

```
- ## Data type : type(variable)로 확인

```python
>>> type(7)
<Class 'int'>
>>> type(3.141592)
<Class 'float'>
>>> type("hi")

```
- ## Variable : value를 담아두며, 연산에 사용 가능
```python
>>> x = 7     #Allocation
>>> print(x)
7
>>> type(x)
<Class 'int'>
>>> x = 9    #Reallocation
>>> y = 81
>>> x * y

```
